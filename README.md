# Statify for Spotify

A web app for viewing your Spotify stats

## Running locally
### Start backend on port 8888
```
cd auth-server/authorization_code/
node app.js
```

### Start frontend on port 3000
```
cd client
npm start
```

### Test in your browser
```
http://localhost:3000/
```
